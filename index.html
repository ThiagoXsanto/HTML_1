A Estrutura básica do HTML

Voltar

O HTML (Hypertext Markup Language) é a linguagem de marcação padrão para criar páginas web. Ele consiste em um conjunto de tags (etiquetas) que são usadas para definir o conteúdo e a estrutura da página. Uma estrutura básica de HTML inclui a tag , que indica o início e o fim da página, a tag , que contém informações sobre a página, como o título da página e metadados, e a tag , que contém o conteúdo visível da página.

Tags no HTML:

Voltar

As tags no HTML são utilizadas para definir o conteúdo e a estrutura da página. Algumas tags comuns incluem p para parágrafos, h1 para títulos, img para imagens, a para links e div para divisões de conteúdo. Cada tag tem sua própria função e utilidade, e as tags podem ser aninhadas para criar estruturas mais complexas.

Executando uma página HTML:

Voltar

Executando uma página HTML: Para executar uma página HTML, você precisará de um navegador web, como o Google Chrome, Firefox ou Safari. Você pode abrir a página HTML salvando o arquivo com a extensão .html em seu computador e abrindo-o usando o navegador. Também é possível executar uma página HTML ao abrir o arquivo usando o navegador.

Formatação de textos no HTML:

Voltar

O HTML oferece várias tags para formatar textos, como para negrito, para itálico, para sublinhar e para riscar. Além disso, também existem tags para criar cabeçalhos, listas e tabelas. É possível também usar CSS para formatar textos e outros elementos.

Links no HTML:

Voltar

Os links no HTML são criados usando a tag (anchor) e o atributo "href", que especifica o endereço da página a ser vinculada. Por exemplo, link cria um link para o endereço "https://www.dio.me" com o texto "link" exibido na página. É possível também criar links para outros arquivos, como imagens e arquivos de áudio e vídeo.
